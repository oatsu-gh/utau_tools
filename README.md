# utau_tools

UTAUのちょっとしたツール

## 開発環境

- Windows 10
- Python 3.8 or Python 3.9
- [utaupy](https://github.com/oatsu-gh/utaupy) 開発版

## ツール紹介

### reclist2region

録音リストをもとに、REAPERのリージョン用CSVを生成する

exe：https://bowlroll.net/file/231585

### scale_otoini

原音設定値をすべて定数倍する

exe：https://bowlroll.net/file/242285
